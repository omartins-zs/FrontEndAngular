

Inicio Preparação do Projeto em Angular 16 e .NET

Comandos

Instalar Angular
npm install -g @angular/cli

npm install -g @angular/cli@16.1.8

Ver a versão do Angular
ng version

Criar projeto

ng new projectangular

Acessar o projeto

cd projectangular

Instalando Extensoes

Angular Schematics

Angular Lancguage

Angular Snippets

Angular Essentials

// Cria o Pages Home
ng g c pages/home

ng g environments

ng generate service services/funcionario

ng g c pages/cadastro

ng g c components/funcionario-form

Explicacao g = Generate | c = component

ng g c pages/editar

ng g c pages/detalhes

Limpar o Cache
npm cache clean --force

Reconstruir o Projeto
ng build

Reiniciar o Servidor de Desenvolvimento
ng serve

ng g c components/excluir
