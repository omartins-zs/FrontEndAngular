Comandos

Instalar Angular
npm install -g @angular/cli

npm install -g @angular/cli@16.1.8

Ver a versão do Angular
ng version

Criar projeto

ng new projectangular

Acessar o projeto

cd projectangular

ng g c pages/editar

Limpar o Cache
npm cache clean --force

Reconstruir o Projeto
ng build
